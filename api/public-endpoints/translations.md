# Translations

## List Translations

### URL

```text
/translations
```

### Parameters

| Name | Type | Description | Required |
| :--- | :--- | :--- | :--- |
| language | string | The code of a language (if you want to get translations in a specific language). | No |

### Response

```json
{
    "data": [
        {
          "code": "en-saheeh-international",
          "name": "Saheeh International",
          "language": "en"
        },
        {
          "code": "ru-kuliev",
          "name": "Kuliev",
          "language": "ru"
        }
    ]
}
```
